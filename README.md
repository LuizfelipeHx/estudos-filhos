# ⚡ Missão Prova

App de estudos gamificado para o 6º ano, com tema de anime. Feito em HTML, CSS e JavaScript puro — sem framework, sem backend.

## Recursos
- 📚 **Aulas** — explicações de cada assunto (com exemplos passo a passo)
- 📋 **Treino** — checklist de domínio em 3 níveis
- 🃏 **Cartas** — flashcards de revisão
- ⚔️ **Batalha** — quiz com perguntas e opções embaralhadas + modo "revisar o que errei"
- 🏅 Ranks de herói, COMBO, XP e contagem regressiva das provas
- 📱 PWA: funciona offline e pode ser instalado no celular

## Como usar
Abra o `index.html` no navegador, ou acesse a versão publicada (GitHub Pages).
